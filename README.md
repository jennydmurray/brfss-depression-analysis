# BRFSS Employment Status and Depression Analysis

This project explores associations between depression and socioeconomic factors in U.S. adults using the 2013 BRFSS survey dataset.

## Research questions
1. Is depression associated with employment status?
2. Does income help explain differences in depression across employment groups?
3. Does education influence the relationship between employment and depression?

## Methods
- Data cleaning and subsetting in R (dplyr)
- Proportion tables and grouped summaries
- Visualization using ggplot2
- Faceted bar charts to compare subgroups

## Data
BRFSS 2013 survey dataset (CDC).  
The raw dataset is not included because the file is too big.

## Outputs
Three visualizations:
- Employment vs depression
- Employment × income vs depression
- Employment × education vs depression

## Tools
R, dplyr, ggplot2
